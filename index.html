BitBoxApp Release Downloads

Release download trends for the BitBoxApp installers.

This dashboard uses GitHub release asset download counts and groups only the app installers by platform. Checksum and signature files are excluded.

Notes

Important context

  • These are download numbers per release, not download numbers over time. Date points are dates releases were made.
  • Does not include iOS App Store or Android Google Play downloads. Only downloads from GitHub.
  • Data taken from https://api.github.com/repos/BitBoxSwiss/bitbox-wallet-app/releases.
